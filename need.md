# QiReader RSS阅读器 产品设计文档

## 1. 产品概述

QiReader 是一款现代化的 RSS 订阅阅读器，提供简洁高效的信息获取体验。支持多订阅源管理、文章分类、标签系统、稍后阅读等功能。

---

## 3. 功能模块详解

### 3.1 左侧边栏 (Sidebar)

#### 3.1.1 顶部功能入口

| 图标 | 功能 | 说明 |
|------|------|------|
| 菜单 | 隐藏侧边栏 | 折叠左侧栏，扩大阅读区域 |
| 用户 | 个人中心 | 用户设置与账户管理 |

#### 3.1.2 订阅分类

```
├── 全部 (未读数量)
├── 稍后阅读
├── 标签
│   └── [用户自定义标签列表]
└── 订阅源 (可展开/折叠)
    ├── 科技新闻 (未读数量)
    ├── 社交媒体 (未读数量)
    └── 游戏 (未读数量)
```

#### 3.1.3 分类操作（右键菜单）

| 操作 | 说明 |
|------|------|
| 置顶 | 将分类固定在顶部 |
| 重命名… | 修改分类名称 |
| 删除 | 删除该分类（红色警示） |

### 3.2 文章列表区

#### 3.2.1 列表头部

- 当前分类名称 + 未读数量
- 「标记以上为已读」按钮
- 视图切换/设置按钮

#### 3.2.2 文章卡片结构

```
┌─────────────────────────────────────────┐
│ [订阅源图标] 订阅源名称        发布时间   │
├─────────────────────────────────────────┤
│ ● 文章标题                               │
│ 文章摘要内容预览，显示前2-3行...          │
│                                    [缩略图] │
└─────────────────────────────────────────┘
```

- 未读文章左侧有圆点标识
- 缩略图右对齐，宽高约 80x60px
- 已读文章标题/摘要颜色变淡

#### 3.2.3 文章操作（右键菜单）

| 操作 | 说明 |
|------|------|
| 在新页面打开源网页 | 跳转到原文链接 |
| 标记为已读 | 切换文章已读状态 |
| 稍后阅读 | 添加到稍后阅读列表 |
| 标签 | 为文章添加标签 |
| 标记以上为已读 | 批量标记该文章及以上所有文章为已读 |

#### 3.2.4 列表底部

- 「显示更多」按钮：加载更多文章

### 3.3 文章详情区

#### 3.3.1 工具栏

| 按钮 | 功能 | 说明 |
|------|------|------|
| ← | 前一篇文章 | 导航到列表中上一篇文章 |
| → | 下一篇文章 | 导航到列表中下一篇文章 |
| 加载网页全文 | 加载原文 | 获取完整网页内容 |
| 稍后阅读 | 收藏 | 添加/移除稍后阅读状态 |
| 标签 | 标签管理 | 为文章添加/移除标签 |
| 标记为未读 | 状态切换 | 切换已读/未读状态 |

#### 3.3.2 文章内容

```
┌─────────────────────────────────────────┐
│ [订阅源名称]                             │
│ 文章标题                                 │
│ 发布时间                                 │
├─────────────────────────────────────────┤
│ 正文内容...                              │
│                                         │
│ [配图]                                   │
│                                         │
│ 正文内容继续...                          │
└─────────────────────────────────────────┘
```

## 4. 交互设计

### 4.1 左键点击行为

| 元素 | 点击行为 |
|------|----------|
| 分类项 | 切换到该分类，更新文章列表 |
| 文章卡片 | 在详情区显示文章内容 |
| 订阅源名称 | 跳转到订阅源详情页 |
| 文章标题 | 在新标签页打开原文 |

### 4.2 右键点击行为

| 元素 | 右键菜单 |
|------|----------|
| 分类项 | 置顶、重命名、删除 |
| 文章卡片 | 打开原文、已读、稍后阅读、标签、标记以上已读 |

### 4.3 状态反馈

- 选中分类：紫色背景高亮 + 左侧竖条
- 已读文章：标题颜色变淡，未读标识消失
- 加载状态：骨架屏/加载动画
- 操作成功：轻微动画反馈

## 6. 数据模型

### 6.1 核心实体

```
User (用户)
├── id
├── username
├── password
└── created_at

Subscription (订阅源)
├── id
├── user_id
├── title
├── feed_url
├── site_url
├── favicon
├── category_id
└── created_at

Category (分类)
├── id
├── user_id
├── name
├── sort_order
├── is_pinned
└── created_at

Article (文章)
├── id
├── subscription_id
├── title
├── content
├── excerpt
├── link
├── thumbnail
├── published_at
├── is_read
├── is_starred
└── created_at

Tag (标签)
├── id
├── user_id
├── name
└── created_at

ArticleTag (文章标签关联)
├── article_id
└── tag_id
```

### 6.2 状态字段

- `is_read`: 文章已读状态
- `is_starred`: 稍后阅读/收藏状态
- `is_pinned`: 分类置顶状态

## 7. API 接口设计

### 7.1 认证

```
POST /api/auth/login     - 用户登录
POST /api/auth/logout    - 用户登出
```

### 7.2 文章

```
GET  /api/articles/all           - 获取所有文章（分页）
GET  /api/articles/starred       - 获取稍后阅读文章
GET  /api/articles/category/:id  - 获取分类下文章
GET  /api/articles/tag/:id       - 获取标签下文章
POST /api/articles/star          - 收藏/取消收藏
POST /api/articles/read          - 标记已读
POST /api/articles/read-above    - 标记以上已读
POST /api/articles/tag           - 添加/移除标签
```

### 7.3 订阅源

```
GET  /api/feed/list              - 获取订阅源列表
POST /api/feed/add               - 添加订阅源
POST /api/feed/edit              - 编辑订阅源
POST /api/feed/delete            - 删除订阅源
POST /api/feed/refresh           - 刷新订阅源（队列）
```

### 7.4 分类

```
GET  /api/categories/list        - 获取分类列表
POST /api/categories/add         - 添加分类
POST /api/categories/edit        - 编辑分类（重命名）
POST /api/categories/delete      - 删除分类
POST /api/categories/pin         - 置顶/取消置顶
```

### 7.5 标签

```
GET  /api/tags/list              - 获取标签列表
POST /api/tags/add               - 添加标签
POST /api/tags/edit              - 编辑标签
POST /api/tags/delete            - 删除标签
```

## 8. 技术栈参考

### 8.1 前端

- **框架**: React / Vue.js
- **UI库**: daisyUI 5 + Tailwind CSS
- **状态管理**: Zustand / Pinia
- **HTTP客户端**: Axios / Fetch

### 8.2 后端

- **框架**: ThinkPHP 8.0
- **认证**: JWT (hulang/think-jwt)
- **DTO**: linron/think-dto
- **模板**: topthink/think-view
- **队列**: kode/queue

### 8.3 数据库

- SQLite / MySQL / PostgreSQL

## 9. 设计规范

### 9.1 色彩

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主色调 | #6366f1 | 紫色，选中/高亮 |
| 强调色 | #3b82f6 | 蓝色，链接/品牌 |
| 危险色 | #ef4444 | 红色，删除/警告 |
| 文字主色 | #1f2937 | 深灰 |
| 文字辅色 | #6b7280 | 中灰 |
| 背景色 | #f9fafb | 浅灰 |
| 卡片背景 | #ffffff | 白色 |

### 9.2 字体

- 标题: 16-24px, Bold
- 正文: 14-16px, Regular
- 辅助: 12-13px, Light

### 9.3 间距

- 基础单位: 8px
- 卡片内边距: 16px
- 列表项间距: 12px
- 区块间距: 24px

### 9.4 圆角

- 按钮: 8px
- 卡片: 12px
- 弹窗: 16px
